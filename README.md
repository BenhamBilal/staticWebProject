
# TechShop Webshop

Statische HTML/CSS webshop voor techproducten.

# Frontend (klant)

 -	Homepage met logo, navigatie, intro-banner en productgrid (≥6 producten)
 -  Productdetailpagina: foto, prijs, voorraad, beschrijving, specificaties, reviews

# Backend (admin)

 - Login/dashboard
 - Productoverzicht in tabel
 - Formulier om producten toe te voegen of aan te passen

# Technisch
 - HTML5 & semantische tags
 - CSS met Flexbox
 - Responsive voor desktop & mobiel
 - Externe CSS + Google Fonts

# Gebruik
 1.	Open index.html voor klantzijde
 2.	Open /admin voor beheer
 3.	Bekijk in browser
